# Installation de hornet-js-builder

- L’installation du builder peut se faire de deux façons :
    - Depuis npm : $ npm install -g hornet-js-builder
    - Depuis les sources :
        - Récupérer les sources de hornet-js-builder depuis le repo git: https://github.com/diplomatiegouvfr/hornet-js-builder
        - Faire une installation globale en étant à la racine de hornet-js-builder : npm install -g
    - Créer un lien symbolique vers le builder dans $HOME/bin depuis votre répertoire global npm (ex : $HOME/.npm-global/bin/hb) pour vous simplifier l'utilisation.
